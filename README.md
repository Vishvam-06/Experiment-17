# Experiment-17

## Aim
To implement and understand basic charts and visual encoding techniques using Python data visualization libraries to analyze sample student performance data.

## Theory
Data visualization is the graphical representation of data to help uncover trends, patterns, and correlations. In this experiment, Python is utilized for data manipulation and plotting:

Pandas: Used to structure raw data (such as Study Hours, Marks, Attendance, Sleep Hours, and Assignments Completed across different Days) into a readable DataFrame format.

Matplotlib and Seaborn: Utilized to create visualizations and apply visual encodings like colors, markers, legends, and grid lines.

The following chart types were implemented to understand different data perspectives:

Data visualization is the graphical representation of information and data. By using visual elements like charts, graphs, and maps, data visualization tools provide an accessible way to see and understand trends, outliers, and patterns in data. In Python, libraries like Pandas are used for data manipulation, while Matplotlib and Seaborn are utilized to render the visualizations and apply visual encodings (colors, markers, legends).

Understanding the appropriate type of graph to use is critical for effective data communication. Below are the fundamental types of graphs used in data analysis:

1. Line Chart
Purpose: Used to display information as a series of data points called 'markers' connected by straight line segments.

Best For: Tracking changes or trends over periods of time (e.g., analyzing Study_Hours over Days).

2. Bar Chart
Purpose: Presents categorical data with rectangular bars with heights or lengths proportional to the values that they represent.

Best For: Comparing different groups or tracking changes over time when the changes are significant (e.g., comparing Marks across different Days).

3. Scatter Plot
Purpose: Uses dots to represent values for two different numeric variables. The position of each dot on the horizontal and vertical axis indicates values for an individual data point.

Best For: Observing relationships, correlations, or patterns between two continuous variables (e.g., determining if higher Study_Hours correlate with higher Marks).

4. Histogram
Purpose: An approximate representation of the distribution of numerical data. It groups continuous data into distinct bins or ranges.

Best For: Understanding the frequency distribution and underlying shape of a single set of continuous data (e.g., the distribution of scores in a class).

## Conclusion
The experiment successfully demonstrated the application of Python libraries (Pandas, Matplotlib, Seaborn) to generate fundamental charts and apply visual encodings. By implementing line charts, bar charts, and scatter plots, raw tabular data was transformed into readable graphical representations.
